# Dmtr-p Tools

## How do I install these formulae?

`brew install dmtr-p/tools/<formula>`

Or `brew tap dmtr-p/tools` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "dmtr-p/tools"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
